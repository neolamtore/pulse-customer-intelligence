# PULSE — Customer Intelligence Dashboard

PULSE is a Python-based ecommerce customer intelligence dashboard designed to analyse customer sentiment, churn risk, behavioural volatility and regional performance trends.

Built to demonstrate how data-driven systems can support operational decision-making inside modern ecommerce businesses.

## Features

- Customer sentiment trend analysis
- Churn risk modelling
- Product-level volatility tracking
- Regional performance benchmarking
- Review density heatmaps
- Revenue vs satisfaction analysis

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy

## Run locally

```bash
pip install -r requirements.txt
python3 main.py
```

Output:

`outputs/pulse_dashboard.png`

## Business Applications

This dashboard demonstrates how lightweight analytics systems can identify:

- customer retention risks
- satisfaction instability
- operational performance gaps
- revenue-performance relationships
